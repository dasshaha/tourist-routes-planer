# tourist-routes-planer

1.1. Цуркина Дарья Денисовна
1.2. Группа Д-Э306

1.3 Тема — "Информационная система для подбора туристических маршрутов (TripPlaner)".
1.4 Цель — "Разработать прототип информационной системы, помогающей пользователям находить персонализированные туристические маршруты на основе заданных критериев".

1.5 Задачи:
1) Проанализировать предметную область.
2) Спроектировать логическую структуру базы данных.
3) Разработать модели бизнес-процессов (BPMN) и потоков данных (DFD).
4) Создать статичный прототип пользовательского интерфейса.
5) Оформить проектную документацию.

1.6 Аннотация: "TripPlaner — это веб-система, позволяющая пользователям находить готовые маршруты для путешествий. Пользователь указывает свои пожелания, система выполняет поиск в каталоге. В системе предусмотрены роли: Гость (ищет маршруты), Автор (создает маршруты), Администратор (управляет контентом)."
